# Sales Performance Analysis

## Objective
Analyze sales data to identify trends, top products, and revenue drivers.

## Dataset
Source: Company sales dataset (anonymized / sample data).

## Tools Used
- Excel
- Power BI

## Key Insights
- Revenue increased by 18% in Q3.
- Top 20% of products generated 65% of total revenue.
- Region A underperformed compared to Region B.

## Recommendation
Focus marketing efforts on high-performing products and optimize inventory in low-performing regions.
